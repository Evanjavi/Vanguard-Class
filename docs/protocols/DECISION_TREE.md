## Handoff Triggers

### Exit Opus When:
- You need to read code
- You need to execute changes
- You need to load files

### Exit Gemini When:
- Strategic decision required
- Visual polish needed
- Security review required

### Exit Claude When:
- Architectural change needed
- Logic bug discovered
- Security concern found

### Exit Ana When:
- Implementation needed
- Visual work needed
- Analysis complete
