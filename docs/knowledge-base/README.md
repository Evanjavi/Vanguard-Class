# Knowledge Base Index

> Captured lessons, patterns, and anti-patterns from project experience.

## Categories

### Lessons Learned
Hard-won insights from real project work.
- Location: `lessons/`
- Format: `LESSON_[TOPIC].md`

### Patterns (Do This)
Proven approaches that work.
- Location: `patterns/`
- Format: `PATTERN_[NAME].md`

### Anti-Patterns (Avoid This)
Mistakes and their solutions.
- Location: `anti-patterns/`
- Format: `ANTI_[NAME].md`

## Quick Add

To add a new lesson:
1. Create file in appropriate folder
2. Use template from `docs/templates/LESSON_TEMPLATE.md`
3. Tag with relevant categories
